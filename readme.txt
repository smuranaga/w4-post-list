== Description ==
Now with more flexibility with Version 1.1.
With W4 development's W4 post list, you can show your category/post list in your themes widget area. After activating plugin, you can Visit the widget area for showing the list in sidebar widget. You can include or exclude posts from inside the post edit page or from the wp-admin post page( From the hover flying menu.ex: edit | quick edit | trash | view | include in W4 post. See Screenshots) .Listings pages and more customizable options will be available through later version. Feel free to reply us @ http://w4dev.com/w4-plugin/w4-post-list/ page if you found any bugs or you have a wonderful suggestion.

== Installation ==
1. Upload zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
